This demonstrates how to update the contents of Breadcrumb items in two different scenarios. The top breadcrumb item is updated via the Breadcrumb API, which is preferred if you are taking advantage of the IDS Enterprise Breadcrumb component API. The bottom breadcrumb item is CSS-only, and is updated directly with string interpolation.
