When a nested dialog is opened from inside the About dialog, it should be possible to close the About dialog using the "X" button. To test:

  1. 1. Click "About" to open the dialog
  2. 2. Click "OPEN NESTED" inside the dialog to open a nested dialog.
  3. 3. Close the nested dialog
  4. 4. Try to close the About dialog. It should close with no issues.
